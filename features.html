Feature 04

SEO対策を標準装備

Google検索で見つけてもらうための技術対策を、すべて標準で搭載しています。

構造化データ（JSON-LD）5種

LocalBusiness・Product・FAQPage・ItemList・BreadcrumbListの5種類のJSON-LDを自動挿入。Google検索結果にリッチリザルトとして表示されます。

XMLサイトマップ

sitemap.xml と robots.txt を自動生成。Google に全ページを漏れなくインデックスさせます。

OGP / Twitter Card

SNSでシェアされた際に画像・タイトル・説明文が美しく表示されるメタタグを自動設定。

メタタグ自動最適化

title・description・canonical タグを各ページの内容に応じて自動生成。

モバイルフレンドリー

全ページ完全レスポンシブ対応。Googleのモバイルファースト指標にも最適化。

Core Web Vitals対応

Googleのページ体験指標（LCP・FID・CLS）を高水準でクリア。PageSpeed 95+を目指します。

パンくずナビ

HTML + JSON-LD BreadcrumbListによるパンくずナビを全ページに自動設置。Google検索結果でサイト構造が表示されます。

代表者名・犬舎名SEO

代表者名や犬舎名・猫舎名で指名検索されたときにヒットするよう、メタタグと構造化データを最適化。

Feature 05

万全のセキュリティ

大切なお客様の情報と、ブリーダー様の個人情報を守ります。

AES-256-GCM 暗号化

個人情報（氏名・メール・電話番号・住所・登録番号）を軍事レベルの暗号化で保護。

EXIF情報自動除去

アップロード画像からGPS位置情報を自動削除。撮影場所がサイト上で公開されるのを防止。

SSL/HTTPS 標準対応

全ページSSL対応。通信を暗号化し、訪問者のブラウザに「安全」と表示されます。

Cloudflare DDoS保護

世界最大級のCDNネットワークが不正アクセスからサイトを保護。安定した稼働を実現。
