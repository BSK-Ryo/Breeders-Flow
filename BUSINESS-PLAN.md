## 1. 事業概要

### サービス概要

**Breeders Flow** は、ブリーダー（犬舎・猫舎）向けの **HP自動生成・運用プラットフォーム**。

- ブリーダーは **LINEで子犬・子猫情報を入力するだけ** で、プロフェッショナルな16ページ以上のWebサイトが自動生成・デプロイされる
- WordPress不要、PC不要、スマホのみで運用可能
- 法的に必要なページ（動物取扱業者表記・特商法・プライバシーポリシー等）も自動生成
- 3つのプロフェッショナルテーマ（プレミアム/モダン/ウォーム）から選択可

### ターゲット市場

**日本国内の犬猫ブリーダー（推定5,000〜15,000事業者）**

| 課題 | 現状 | Breeders Flowの解決策 |
|------|------|----------------------|
| HP制作費が高い | 制作会社に30〜100万円 | 初期費用0円（キャンペーン） |
| 更新が面倒 | PCスキル必要、メール依頼で数日待ち | LINEで即時更新 |
| ポータル手数料 | 1件あたり15〜30%の仲介手数料 | 自社HP直販で手数料0% |
| 差別化が困難 | ポータル上で横並び | 独自ブランドサイトで信頼構築 |
| 法的対応が不安 | 法改正への追随が困難 | 法的ページ自動生成・更新 |

## 2. アウトバウンド営業KPI（SNS DM中心）

### 営業ファネル

```
リスト作成（見込みブリーダー調査）
  ↓
フォロー・いいね（関係構築・ウォームアップ）
  ↓
DM送信（初回アプローチ）
  ↓
DM返信（興味あり）
  ↓
無料相談アポ（ZOOM / LINE通話）
  ↓
デモ実施（実際のサイト例を提示）
  ↓
契約成立
```

### 行動量KPI

創業者が開発と営業を兼務する前提。**営業に1日2〜3時間**を確保。

| 指標 | 日次 | 週次 | 月次 |
|------|------|------|------|
| **リスト追加**（見込み客調査） | 10件 | 50件 | 200件 |
| **フォロー/いいね**（ウォームアップ） | 15件 | 75件 | 300件 |
| **DM送信**（初回アプローチ） | 10〜15件 | 50〜75件 | 200〜300件 |
| **DM返信対応** | 随時 | - | - |
| **相談アポ実施** | - | 2〜3件 | 8〜12件 |
| **デモ実施** | - | 1〜2件 | 5〜8件 |

### 転換率KPI

| ステージ | 目標 | 補足 |
|----------|------|------|
| DM送信 → 返信 | **10〜20%** | 業界平均10〜15%。ウォームアップで向上 |
| 返信 → 相談アポ | **30〜40%** | 返信=興味ありのため高転換率 |
| 相談アポ → デモ | **80%** | 日程調整のみ |
| デモ → 契約 | **25〜35%** | 初期30%目標。事例蓄積で向上 |

### ファネル試算（月次ベース）

```
DM送信 250件/月
  × 返信率 15%     = 37件（返信）
  × アポ率 35%     = 13件（相談アポ）
  × 実施率 80%     = 10件（デモ実施）
  × 成約率 30%     = 3件（新規契約）
```

> 月3社獲得が基本シナリオ。ファネル改善で4〜5社/月も射程圏内。

| **市場規模の限界** | 高 | 中 | TAM 5,000社以上。1%でも50社。ペットショップ・トリマー等への横展開オプション |
| **大手参入** | 高 | 低 | ブリーダー特化の深さが参入障壁。LINE連携・法的ページ自動生成は汎用サービスでは困難 |
| **解約率の上昇** | 高 | 中 | 6ヶ月最低契約＋高スイッチングコスト（サイト移行の手間）＋SEO効果の蓄積で継続インセンティブ |
| **開発リソース不足** | 中 | 高 | MVPに集中し、機能追加は顧客フィードバック優先で絞り込む |
| **営業リソース不足** | 中 | 高 | SEO・紹介プログラムでインバウンド比率を上げる。セルフサーブ申込導線を整備 |
| **規制変更** | 中 | 低 | 動物愛護法の改正動向を監視。法的ページの柔軟な更新機能で迅速対応 |
| **Cloudflare依存** | 低 | 低 | 静的HTML出力のため他プラットフォームへの移行は容易 |

## 10. 補助金活用戦略

### 自社が申請可能な補助金

#### デジタル化・AI導入補助金 2026（旧IT導入補助金）

2026年度より「IT導入補助金」から名称変更。

| 項目 | 内容 |
|------|------|
| **対象** | 中小企業・小規模事業者のIT導入支援 |
| **補助率** | 原則1/2以内（一定条件で2/3） |
| **補助額** | 5万〜450万円 |
| **公募時期** | 2026年春頃より順次開始 |
| **自社活用** | IT導入支援事業者として登録し、顧客（ブリーダー）のBreeders Flow導入を支援する形で申請 |

#### 新事業進出補助金（旧事業再構築補助金の後継）

事業再構築補助金は第13回（2025年3月）で終了。後継として「新事業進出補助金」が継続。2026年度は「新事業進出・ものづくり補助金（仮称）」に再編予定。

| 項目 | 内容 |
|------|------|
| **対象** | 新分野展開・業態転換等に取り組む中小企業 |
| **補助率** | 1/2〜3/4 |
| **補助額** | 最大9,000万円 |
| **自社活用** | 自社の新規SaaS事業としての開発費・人件費の補填 |

#### 小規模事業者持続化補助金

| 項目 | 内容 |
|------|------|
| **対象** | 販路開拓の取り組み |
| **補助率** | 2/3（最大50万〜200万円） |
| **顧客活用** | ブリーダー顧客がHP制作費として申請可能 → 導入ハードル低下 |

### 補助金活用の2軸戦略

1. **自社申請（開発費補填）**
   - 新事業進出補助金で開発費・人件費の一部を補助
   - ソフトウェア開発費、クラウドインフラ費が対象経費

2. **顧客側申請支援（営業ツール）**
   - デジタル化・AI導入補助金 → 顧客のBreeders Flow導入費を補助
   - 持続化補助金 → 顧客のHP制作・販路開拓費を補助
   - **「実質負担0円〜半額」の訴求が可能** → 営業の強力な武器に

## 付録: 用語集

| 用語 | 説明 |
|------|------|
| **MRR** | Monthly Recurring Revenue（月次経常収益） |
| **ARR** | Annual Recurring Revenue（年間経常収益）= MRR × 12 |
| **ARPU** | Average Revenue Per User（1社あたり平均月額売上） |
| **LTV** | Lifetime Value（顧客生涯価値）= ARPU × 平均契約期間 |
| **CAC** | Customer Acquisition Cost（顧客獲得単価） |
| **NRR** | Net Revenue Retention（売上維持率） |
| **NPS** | Net Promoter Score（顧客推奨度スコア） |
| **TAM** | Total Addressable Market（獲得可能な最大市場規模） |
| **PMF** | Product-Market Fit（プロダクトと市場の適合） |
| **Churn Rate** | 解約率（月次で解約した割合） |
