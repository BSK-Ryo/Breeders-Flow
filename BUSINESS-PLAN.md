## 1. 事業概要

### サービス概要

**Breeders Flow** は、ブリーダー（犬舎・猫舎）向けの **HP自動生成・運用プラットフォーム**。

- ブリーダーは **LINEで子犬・子猫情報を入力するだけ** で、プロフェッショナルな16ページ以上のWebサイトが自動生成・デプロイされる
- WordPress不要、PC不要、スマホのみで運用可能
- 法的に必要なページ（動物取扱業者表記・特商法・プライバシーポリシー等）も自動生成
- 3つのプロフェッショナルテーマ（プレミアム/モダン/ウォーム）から選択可

### ターゲット市場

**日本国内の犬猫ブリーダー**

| 課題 | 現状 | Breeders Flowの解決策 |
|------|------|----------------------|
| HP制作費が高い | 制作会社に30〜100万円 | 初期費用0円（キャンペーン） |
| 更新が面倒 | PCスキル必要、メール依頼で数日待ち | LINEで即時更新 |
| ポータル手数料 | 1件あたり15〜30%の仲介手数料 | 自社HP直販で手数料0% |
| 差別化が困難 | ポータル上で横並び | 独自ブランドサイトで信頼構築 |
| 法的対応が不安 | 法改正への追随が困難 | 法的ページ自動生成・更新 |

### 市場規模（TAM / SAM / SOM）

| 区分 | 定義 | 推定値 | 根拠 |
|------|------|--------|------|
| **TAM** | 日本の犬猫繁殖業者（第一種動物取扱業「販売」登録） | **約10,000〜13,000事業所** | 環境省「犬猫等販売業のうち繁殖を行う者」12,603件（2016年）。2019年法改正以降の規制強化で減少傾向にあり、現在は10,000前後と推定 |
| **SAM** | そのうちSNS（Instagram等）で活動しておりリーチ可能な層 | **約3,000〜5,000事業所** | TAMの30〜50%。Instagramでハッシュタグ検索可能なアクティブブリーダー |

> **注意:** TAMの精緻化には環境省最新データ（令和5年度版）の確認が必要。ローンチ前に一次調査を推奨。

### ポジショニング戦略

**競争軸:**

```
                  ブリーダー特化度（高）
                        ↑
                        │
    ブリーダー専門業者   │  ★ Breeders Flow
    （高価格・PC必須）   │  （中価格・LINE完結）
                        │
  ───────────────────────┼──────────────────── 使いやすさ（高）→
                        │
    制作会社             │  汎用HPサービス
    （高価格・高品質）   │  （低価格・汎用）
                        │
```

**勝負する軸:** 「使いやすさ」（LINE完結・PC不要）× 「ブリーダー特化」
**勝負しない軸:** 価格の安さ（Wix等の汎用サービスには勝てない）、デザインの自由度（制作会社には勝てない）

**コアメッセージ（1行）:** 「LINEで入力するだけ。あなたの犬舎専用HPが、今日できる。」

**差別化の持続性:**
- **短期の壁:** ブリーダー業界特化のテンプレート・法的ページ自動生成（汎用サービスが真似しにくい）
- **中期の壁:** ブリーダー顧客基盤とその成功事例の蓄積（ネットワーク効果）
- **長期の壁:** ブリーダー業務データの集積によるサービス拡張（予約管理・顧客管理等）

### 成長フライホイール

**フライホイール①: SEO成功事例ループ**

```
顧客のHPが検索上位に表示
        ↓
「犬舎名+地域」で1位の成功事例ができる
        ↓
事例をDM営業に活用（「○○犬舎様は3ヶ月で検索1位に」）
        ↓
DM成約率が向上（事例なし10% → 事例あり25%以上を目標）
        ↓
顧客が増える → 成功事例がさらに増える → ...
```

**フライホイール②: 紹介ループ**

```
顧客が満足（HPから問い合わせが来る体験）
        ↓
ブリーダー仲間との会話で自然に紹介
        ↓
紹介経由で新規獲得（CACはDMの1/5以下）
        ↓
利益率向上 → 紹介インセンティブに投資可能
        ↓
紹介プログラム強化 → 口コミ加速 → ...
```

**フライホイール③: コンテンツループ（中期）**

```
顧客サイトのデータ蓄積（人気犬種、問い合わせ傾向等）
        ↓
ブリーダー向けノウハウコンテンツに変換
        ↓
SNS/ブログで発信 → オーガニック流入増
        ↓
インバウンドリードが増加 → DM依存度が下がる → ...
```

> **線形成長からの脱却:** 1年目はDM主体の線形成長（月+3社）だが、フライホイールが回り始める6ヶ月後以降、複利的に成長が加速する設計。2年目のインバウンド比率50%がこの転換点の目安。

## 2. アウトバウンド営業KPI（SNS DM中心）

### 営業ファネル

```
リスト作成（見込みブリーダー調査）
  ↓
フォロー・いいね（関係構築・ウォームアップ）
  ↓
DM送信（初回アプローチ）
  ↓
DM返信（興味あり）
  ↓
無料相談アポ（ZOOM / LINE通話）
  ↓
デモ実施（実際のサイト例を提示）
  ↓
契約成立
```

### 行動量KPI

創業者が開発と営業を兼務する前提。**営業に1日2〜3時間**を確保。

| 指標 | 日次 | 週次 | 月次 |
|------|------|------|------|
| **リスト追加**（見込み客調査） | 10件 | 50件 | 200件 |
| **フォロー/いいね**（ウォームアップ） | 15件 | 75件 | 300件 |
| **DM送信**（初回アプローチ） | 10〜15件 | 50〜75件 | 200〜300件 |
| **DM返信対応** | 随時 | - | - |
| **相談アポ実施** | - | 2〜3件 | 8〜12件 |
| **デモ実施** | - | 1〜2件 | 5〜8件 |

### 転換率KPI

| ステージ | 保守シナリオ | 基本シナリオ | 補足 |
|----------|:-----------:|:-----------:|------|
| DM送信 → 返信 | **7%** | **15%** | 冷たいDMは3〜8%が相場。ウォームアップ（事前フォロー・いいね）で10〜15%に向上 |
| 返信 → 相談アポ | **25%** | **35%** | 返信=興味ありのため比較的高い。ただし「情報収集だけ」層もいる |
| 相談アポ → デモ | **70%** | **80%** | ブリーダーは多忙なため、日程が合わずノーショーもある |
| デモ → 契約 | **20%** | **30%** | 初期は事例不足で低め。事例蓄積で向上 |

> **注意:** 基本シナリオのDM返信率15%は「事前にフォロー・いいね・コメントで関係構築済み」が前提。初回接触でいきなりDMを送る場合、返信率は5%前後まで下がる可能性がある。

### ファネル試算（月次ベース・2シナリオ）

**基本シナリオ（ウォームアップあり・DM返信率15%）:**
```
DM送信 250件/月
  × 返信率 15%     = 37件（返信）
  × アポ率 35%     = 13件（相談アポ）
  × 実施率 80%     = 10件（デモ実施）
  × 成約率 30%     = 3件（新規契約）
```

**保守シナリオ（ウォームアップ不足 or 初期フェーズ・DM返信率7%）:**
```
DM送信 250件/月
  × 返信率 7%      = 17件（返信）
  × アポ率 25%     = 4件（相談アポ）
  × 実施率 70%     = 3件（デモ実施）
  × 成約率 20%     = 0.6件（新規契約）→ 月1件がやっと
```

> **重要:** 保守シナリオでは月1社がやっとであり、基本シナリオの月3社とは大きな乖離がある。ローンチ前のβテスト期間（2月）に50〜100件のDMを送り、実測値を取ることを強く推奨。実測値に基づいて計画を修正すること。

## 7. ユニットエコノミクス

| 指標 | 値 | 算出根拠 |
|------|-----|---------|
| **ARPU** | ¥17,000/月 | Standard×80% + Premium×20%（割引後加重平均） |

| **CAC（6ヶ月後目標）** | ¥37,500〜¥50,000 | テンプレート化・事例蓄積で15〜20時間に短縮 |

> **評価**: 初期のLTV/CAC 3.0〜3.3倍はSaaS健全基準（3倍以上）をぎりぎりクリアする水準。楽観的ではないが持続可能。CAC回収期間5〜6ヶ月は6ヶ月最低契約期間内に収まるため、解約リスクに対するバッファは薄い。**紹介経由の獲得比率を6ヶ月後に20%以上に引き上げることが、ユニットエコノミクス改善の最重要レバー。**

| **補助額** | 5万〜450万円 |
| **公募時期** | 2026年春頃より順次開始 |
| **自社活用** | IT導入支援事業者として登録し、顧客（ブリーダー）のBreeders Flow導入を支援する形で申請 |

#### 新事業進出補助金（旧事業再構築補助金の後継）

事業再構築補助金は第13回（2025年3月）で終了。後継として「新事業進出補助金」が継続。2026年度は「新事業進出・ものづくり補助金（仮称）」に再編予定。

| 項目 | 内容 |
|------|------|
| **対象** | 新分野展開・業態転換等に取り組む中小企業 |
| **補助率** | 1/2〜3/4 |
| **補助額** | 最大9,000万円 |
| **自社活用** | 自社の新規SaaS事業としての開発費・人件費の補填 |

#### 小規模事業者持続化補助金

| 項目 | 内容 |
|------|------|
| **対象** | 販路開拓の取り組み |
| **補助率** | 2/3（最大50万〜200万円） |
| **顧客活用** | ブリーダー顧客がHP制作費として申請可能 → 導入ハードル低下 |

### 補助金活用の2軸戦略

1. **自社申請（開発費補填）**
   - 新事業進出補助金で開発費・人件費の一部を補助
   - ソフトウェア開発費、クラウドインフラ費が対象経費

2. **顧客側申請支援（営業ツール）**
   - デジタル化・AI導入補助金 → 顧客のBreeders Flow導入費を補助
   - 持続化補助金 → 顧客のHP制作・販路開拓費を補助
   - **「実質負担0円〜半額」の訴求が可能** → 営業の強力な武器に

## 付録: 用語集

| 用語 | 説明 |
|------|------|
| **MRR** | Monthly Recurring Revenue（月次経常収益） |

| **NRR** | Net Revenue Retention（売上維持率） |

| **Churn Rate** | 解約率（月次で解約した割合） |
